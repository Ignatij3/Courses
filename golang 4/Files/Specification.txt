func Calculate(data [][]float64) (res []float64, ok bool)

Функция Calculate принимает квадратную систему линейных алгебраических уравнений в виде переменной [][]float64, в ней должны находиться n слайсов, в каждом из которых
находятся n коэффициентов при неизвестных и решение уравнения. Функция возвращает слайс с полученными значениями неизвестных и bool, true - система имеет единственное решение,
false - иначе, или при некорректных входных данных.

Входной файл:
В первой строке находится кол-во последующих строк (n). Далее идёт квадратная система линейных алгебраических уравнений, состоящая из n уравнений. В каждой строке находится
n+1 чисел, разделённых пробелом, первые n являются коэффициентами при неизвестных, а n+1-е - решение уравнения.
